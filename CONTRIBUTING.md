# Contributing to DrinkReminder

感谢您有兴趣为 DrinkReminder 做出贡献！

## 如何贡献

### 报告 Bug

如果您发现了 bug，请通过 [GitHub Issues](../../issues) 提交报告。请包含：

- 详细的描述
- 复现步骤
- 预期行为
- 实际行为
- 系统环境（Windows 版本、.NET 版本等）

### 提交代码

1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/amazing-feature`)
3. 提交更改 (`git commit -m 'feat: add some amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建 Pull Request

### 代码规范

- 遵循 C# 编码规范
- 使用有意义的变量和方法命名
- 添加必要的注释
- 确保代码通过所有测试

### 提交信息规范

使用 Conventional Commits 格式：

- `feat:` 新功能
- `fix:` Bug 修复
- `docs:` 文档更新
- `style:` 代码格式调整
- `refactor:` 重构
- `test:` 测试相关
- `chore:` 构建/工具相关

## 开发环境设置

1. 安装 [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
2. 克隆仓库
3. 运行 `dotnet restore` 还原依赖
4. 运行 `dotnet build` 构建项目

## 许可证

通过贡献代码，您同意您的代码将以 MIT 许可证发布。
